HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • "li" shows an unordered list of items
  • Each "li" represents a new bullet point
  • "id" is used to describe the relation with the content

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The class should be connected in front of each 'section'
  • All the codes related to CSS are locaeted at '#style.css'
  • Student prework 3.2.5 to "Add Style Using Class Atributes"

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds modifications in the current working branch to the staging area, it indicates that we want to add all changes
  • git commit -m "add starter code": add a commit to the change that will be pulled over
  • to push the changes into GitHub use he sequence: " git pull origin main" , then move to "git push origin feature/starter-code"

  • arrays: var = variable (declaration[0], declarion[1], etc......)
